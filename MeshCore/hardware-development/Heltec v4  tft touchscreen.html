Hi all, just a little project I was working on. I wanted to share and maybe for people with better coding skills then me, to give them a shot to make it even better. I wanted to basically make a DIY version of the heltec v4 expansion kit. But simpler. More leaning towards the traditional ble compagnion. I was hoping I could just run the ripple firmware. But I could not make the touch input work, probably because the gpio pins are hardcoded and I cannot change what is necessary to make it work without the extension board from heltec. So I took a different approach. I (relying on AI) adjusted the heltec v4 tft firmware to be able to use a touchscreen and build a simple gui. Simply because i could define the used gpio pins. Thing is working now, still all the same functionality as the original CP. My goal is to either get ripple working, or make something so I could type messages on the screen to be able to use the CP as a standalone or a ble compagnion. Here you can find my barbaric version of the firmware. For testing. https://github.com/Quark1980/Meshcore-Touch (edited)
