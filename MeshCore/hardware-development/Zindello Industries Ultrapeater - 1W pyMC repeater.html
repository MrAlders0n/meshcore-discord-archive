📚 Forums / hardware-development / Zindello Industries Ultrapeater - 1W pyMC repeater
zindello (VK2MES) 2026-04-15 5:00 p.m.
I've been working on a new product. It's a LuckFox Pico Ultra based E22P 1W (30dbm) shield. The device runs pyMC and has a lot of options. pyMC allows you to:
  • Run a MeshCore repeater
  • LetsMesh/Meshmapper observers
  • Custom MQTT integrations
  • Room Server
  • TCP companions (connect over wifi on your network)
The boards are currently available with an E22 900M30S radio (covers 868/915 but an external filter is recommended in noisy environments) or an E22915M30S (Built-in filter for use in the the 915MHz band). I've wrapped it up with a software install guide, accessories, 3d printed case (or printable, the file is in the repo), along with other accessories in the store. We now have a few of these running in Aus, the UK and NZ, it would be great to see the product grow. I'd appreciate support from anyone who wants one. I have a few left in stock right now before I get the next batch done. I ship worldwide. https://zindello.com.au/ultrapeater (edited)
