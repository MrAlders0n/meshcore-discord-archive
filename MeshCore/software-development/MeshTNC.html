Dan - WI7FI [PNW] 2025-11-04 4:54 a.m.
MeshTNC is a tool for easily piping raw data to and from consumer grade LoRa radios. MeshTNC is based on Meshcore, and supports most of the same hardware. The README of the project goes into more details about what you can do with this, but tl;dr
  • APRS clients that can use KISS modems will work with MeshTNC in KISS mode
  • Linux Kernel driver support in KISS mode
    • You can just do APRS, AX.25, or Ethernet directly over LoRA
    • You can packet capture raw data on any modem preset and syncword supported by your radio by kissattaching a MeshTNC and running tcpdump -i ax0
  • You can send raw bytes on any radio preset using serial commands and encoding your data as hex. It can also log received packets as hex over serial.
Project homepage: https://github.com/datapartyjs/MeshTNC Project roadmap: https://github.com/orgs/datapartyjs/projects/4 (edited)
