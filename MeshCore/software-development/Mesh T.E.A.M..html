I've been working on a backcountry team app for android that uses MeshCore infrastructure for keeping track of group members. It was created with hunting and backcountry recreation in mind, but I'm sure it would be useful for other things too. It has messaging via public channels, private channels and direct messaging, and a map system that allows for offline use, tracking of group members and waypoint sharing. It is also intended to color code users on the map based on the hops for real time SA of connectivity. I've hosted the apk in the repo here. https://github.com/tmacinc/meshcore-team-alpha Keep in mind that it's alpha, but I'm looking for feedback on bugs or potential feature that people would like to see. I'm also debating making it open source if there's some interest in helping out. One thing I'm likely unable to do alone is create version for iOS. Team is the working title and I'm open to input.
